# Site Blog

#### 项目介绍
轻量级简洁个人博客系统 

> 此项目借鉴[Tale](https://gitee.com/biezhi/tale/)，原项目使用了轻量级 mvc 框架 [Blade](https://github.com/biezhi/blade) 开发

演示站点：https://yangxs.ink

#### 软件架构

SpringBoot+Thymeleaf+Mysql+myBatis

#### 此版本

1.文章发布、修改、删除
2.文章留言评论、回复
3.后台管理相关
